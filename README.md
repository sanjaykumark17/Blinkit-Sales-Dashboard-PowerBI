## Project Overview
This project analyzes Blinkit sales data using Power BI to provide business insights through interactive dashboards.

## Tools Used
- Power BI
- SQL
- Power Query
- DAX

## Dashboard Features
- Total Sales
- Total Orders
- Product Category Analysis
- Outlet Size Analysis
- Sales by Location
- Interactive Filters
