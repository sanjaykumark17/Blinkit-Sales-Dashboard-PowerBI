# 🛒 Blinkit Sales Dashboard | Power BI

## 📌 Project Overview
This project analyzes Blinkit's sales data using Power BI to provide actionable business insights through an interactive dashboard.

## 🚀 Tools Used
- Power BI
- SQL
- Power Query
- DAX

## 📊 Dashboard Features
- Total Sales Analysis
- Sales by Product Category
- Outlet Size Analysis
- Outlet Location Analysis
- KPI Cards
- Interactive Filters & Slicers

## 💡 Key Insights
- Identified top-performing product categories.
- Compared sales across outlet sizes and locations.
- Built interactive visualizations for business decision-making.
